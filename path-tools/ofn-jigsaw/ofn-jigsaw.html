ofn-jigsaw

Purpose

This script creates jigsaw patterns as paths.

Operation

The script is started from Filters>Render>Path>Jigsaw pattern

Parameters:

  • Rows and Columns are the number of rows and columns to create.
  • Roundness: controls how round the tabs are. A roundness of 0 generates triangles.
  • Randomness: controls how random the tabs are. A randomness of 0 generates identical tabs.
  • Deepness: controls how high/deep the tabs are.
  • Wideness: controls how wide the tabs are.
  • Path: determines if the output is a single path or if horizontal and vertical cuts are on separate paths.

The "*ness" parameters are not absolute values. They are just set so that keeping everything at mid-range (50) generates a decent-looking pattern. Extreme values can generate overlapping cuts.

Examples of use

Bump-mapping a jigsaw pattern in picture:

  • Generate the jigsaw pattern with the script
  • Add a layer, fill with 50 gray (R=G=B=127)
  • Edit>Stroke path on it, in "Line" mode, with Black. The width of the line is dependent on the size of your pattern, as a rule of thumb, use a width of 3 for 100x100 pieces.
  • Blur the lines slightly
  • Use *Filters>Map>Bump map", using your jigsaw grid layer as the "Aux input"
