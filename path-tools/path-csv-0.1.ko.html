path-csv

이 플러그인은 경로를 CSV 파일로 들여오기/내보내기 함. CSV는 엑셀이나 오픈오피스 캘크 프로그램으로의 쉬운 인터페이스임. 들여오기 할 때는 단일 파일에서 여러 경로를 들여오기할 수 있음. 내보내기 할 때는 현재 경로를 내보내기함.

주의할 점

국가별 십진수 형식이 규정된진 것이 없음. 특히, 천 단위 분리자나 소수점이 그러함. 경로 목록 창에서 우클릭해서 호출.

내보내기할 때도 기존 파일이 있어야만 작동함. 따라서, 빈 파일을 미리 만들어 두어야 함. 내보내기할 때는 파일을 덮어씀.

CSV 파일 형식

처음 여덟 칼럼에 데이터를 넣어야 함(A열부터 H열까지):

  • (A) 경로 이름(매 행에서 반복해야 함).
  • (B) 부경로 식별자(매 행 반복).
  • (C) 후방 핸들의 X 좌표
  • (D) 후방 핸들의 Y 좌표
  • (E) 앵커 점의 X 좌표
  • (F) 앵커 점의 Y 좌표
  • (G) 전방 핸들의 X 좌표
  • (H) 전방 핸들의 Y 좌표

1열(A열)이 새로운 이름이면 새 경로의 시작임

2열(B열)이 새로운 식별자이면 새 부경로의 시작됨.

경로 이름은 재사용 불가.

같은 경로 이름 안에서 부경로 이름표 재사용 불가

부경로의 첫 행이 마지막 행과 정확히히 같으면 "닫힌" 경로로 묵시적으로 지정됨.

들여오기 할 때, 닫힌 경로에서 마지막 행은 김프 부경로에서 새 점을 만들지 않고 닫힌 부경로라는 표식을 붙임.

반대로, 내보내기할 때, 닫힌 부경로는 처음 좌표 3쌍 한 벌의 복사본으로 끝나게 해서 지정함.

1열(A열)이나 2열(B열)이 비었거나 3~8열(C~H열)이 수치가 아니면 무시되므로 헤더나 분리자나 주석으로 사용 가능.
