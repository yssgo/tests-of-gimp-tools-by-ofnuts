ofn-remove-tangents

This scripts removes tiny tangents handles that wreak havoc when stroking. Since these tangents are very often accidental and tooo smal to have useful visual effect they can be removed without any negative impact.

For some background information on paths see here.

Use

The script is called from the Paths dialog. Right-click the path you want to clean and invoke Tools>Remove small tangents (towards the bottom of the menu).

The only parameter is a size, which is the size under which a tangent is considered accidental and should be removed (under a pixel, usually).

The script creates a copy of the path, named "Cleaned {source path name}"
