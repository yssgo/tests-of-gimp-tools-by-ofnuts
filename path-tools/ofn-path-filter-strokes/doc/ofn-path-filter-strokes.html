ofn-path-filter-strokes

Purpose

This script removes strokes from a path, based on the "size" of the stroke. This size criterion can be:

  • The number of anchors/segments
  • The length
  • The area (see Notes below)

Usage

Once installed, this script appears in the right-click menu of the Paths list as Tools>Filter strokes on size

The script doesn't delete the original path, it creates a new path which is a copy of the source path with some strokes removed.

The threshold value is tested for equality (the strokes for which the measure is equal to the theshold are kept)

Notes

The computation of the stroke area is approximate, it is the area of the polygon going through the anchor points. The computation is only valid for "simple" (non-self-intersecting) polygons (paths created from a selection are alays of that kind)
