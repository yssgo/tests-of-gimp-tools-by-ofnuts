ofn-save-next

Purpose

This script saves the current image using the next number in sequence, so you don’t have to open the file selector and enter the sequence manually.

Usage

The script appears in the menu as: FileðŸ šSave next.

It is assumed that the file is already named using the scheme {prefix}{number}{.ext}.

The script increases the {number} part by one to save the image. If {number} has leading zeros, the new number is padded with zeros on the left to match the length of the initial number.

You normally open the first file using the regular FileðŸ šOpen, and then use the script to save the image.

The file is saved/exported using the filetype used when opening it. Saving is done using the default options for the file type.

The plugin is best assigned to a keystroke. It appears in the shortcut editor as python-fu-ofn-save-next.

Warning: the script will overwrite any exising file in case of name collision.
