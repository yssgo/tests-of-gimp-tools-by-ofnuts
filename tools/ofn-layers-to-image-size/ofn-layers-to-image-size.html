ofn-layers-to-image-size

This script is the equivalent of applying Layer>Layer to image size to all or all visible layers.

The script is invoked using the menu Image>Resize layers to image size.

The options are:

  • Resize layers: specifies which layers are resized
    • All layers: the script resizes all layers.
    • Visible layers: the script only resizes layers that are actually visible (they are visible and all their parent layers (if any) are also visible).
  • Add alpha: specifies if an alpha channel is added to layers without one. Without this, the resized layer is padded with the background color and hides all layers below it.
  • Skip text: specifies if text layers are explicitly ignored (resizing them makes them uneditable)
