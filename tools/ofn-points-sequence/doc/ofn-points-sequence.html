ofn-points-sequence

Purpose

This script adds sequential number labels to an image, using a path

Operation

  1. Create a path using the Path tool, and click on the places where you want number labels to appear:

  2. Invoke the script to create the labels (Filters>Render>Sequence numbers on anchors)

Parameters

  • Path: the path to use
  • Font, Font size: the font
  • Start: the starting number of the sequence
  • Offset X, Offset Y: a shift of the centers of the labels from the anchors positions. Positive numbers shift down and to the right. If both are 0 the label is centered on the anchor.

Notes

  • Labels are created as individual text layers, so that the position of some can be re-adjusted by hand.
  • If there are several "strokes" in the path, each stroke is considered a separate sequence. If you have stray numbers, look for isolated anchors in your path.
