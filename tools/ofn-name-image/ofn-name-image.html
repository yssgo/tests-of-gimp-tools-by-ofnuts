ofn-name-image

This script is used to give a meaningful name to an image (as displayed in the title bar) without saving the image to disk.

This script is called via the menu entry: Image>Name image

The only parameter is the new image name. As safety measures:

  • The new name is accepted only if the current name is between brackets: [Untitled]. If you expect to rename the image several times, give it a name that starts and ends with brackets: [New Name].
  • The corresponding pseudo-filename (which will be used by Gimp if you attempt to save) corresponds to the "natural" user images directory for your platform (as obtained with gimp.user_directory(4))
